## 🌸 About the Project

This is a **beautiful Birthday Surprise Web App** built using **Python + Streamlit**.  
It is designed to create an emotional and romantic digital birthday experience with animations, music, love letters, and memories.

---

## ✨ Features

🎉 **Animated Birthday Home Page**  
- Stylish glowing title  
- Romantic dark gradient UI  

❤️ **Heart Animation Section**  
- Animated heart visualization  
- Smooth real-time rendering  

💌 **Love Letter Page**  
- Romantic styled message box  
- Emotional personalized letter  

📸 **Memories Gallery**  
- Image slideshow / carousel  
- Click-to-view memory system  
- Smooth transitions  

🎈 **Birthday Celebration Tab**  
- BIG “HAPPY BIRTHDAY TO YOU” title  
- Floating balloons animation  
- Fireworks-style effects  
- Background music integration
